# useScrollPosition

A React [hook] for components that care about
the nearest scrollable container's scroll position.

## Installation

```shell
yarn add @hzcore/scroll-monitor
```

## Usage

```
import {useScrollPosition} from '@hzcore/scroll-monitor';
```

### Signature

```ts
function useScrollPosition(
  handler: (position: ScrollPosition) => void,
): React.RefObject<HTMLElement>;
```

or, with a `providedRef`:

```ts
function useScrollPosition(
  handler: (position: ScrollPosition) => void,
  providedRef: React.RefObject<HTMLElement>,
): void;
```

### handler: `handler: (position: ScrollPosition) => void,`

A function to handle scroll position updates.
Receives the latest `ScrollPosition`.

The `ScrollPosition` object shape is:

```ts
interface ScrollPosition {
  /**
   * The number of pixels the nearest scrollable container
   * is scrolled vertically,
   */
  top: number | null;
  /**
   * The number of pixels the nearest scrollable container
   * is scrolled horizontally,
   */
  left: number | null;
}
```

### providedRef: `providedRef?: React.RefObject<HTMLElement>`

An optional ref object to use.
Useful when the component needs to handle ref forwarding.

See [Usage with Shared Ref / Ref Forwarding] for more.

### returns `React.RefObject<HTMLElement> | void`

If a `providedRef` is passed to `useScrollPosition`,
returns `void`.

If no `providedRef` is passed, returns a `ref` object.
The `ref` should be passed to an underlying DOM node.
Note that the node does not have to be scrollable itself,
as `useScrollPosition` will traverse the DOM to find a scrollable parent
to observe.

## Example

<Playground>
  {() => {
    const ScrollPositionUser = () => {
      const [position, setPosition] = React.useState(null);
      const ref = useScrollPosition(setPosition);
      return (
        <div
          ref={ref}
          style={{
            display: 'flex',
            justifyContent: 'center',
            alignItems: 'center',
            width: '100%',
            height: 250,
            fontSize: '2em',
          }}
        >
          {JSON.stringify(position, null, 2)}
        </div>
      );
    };
    return <ScrollPositionUser />;
  }}
</Playground>

## Usage with Shared Ref / Ref Forwarding

Sometimes a component needs to [forward a ref] to its children,
or [use a ref][useref] with other hooks.
To support these cases, `useScrollPosition`
accepts an optional `providedRef` argument:

```js
const SomeRefForwardingComponent = React.forwardRef((props, forwardedRef) => {
  const [position, setPosition] = React.useState(null);
  // We pass `forwardedRef` (which _must_ be a ref object) to `useScrollPosition`.
  // Note that a `ref` will _not_ be returned.
  useScrollPosition(setPosition, forwardedRef);
  return <div ref={forwardedRef}>{JSON.stringify(position, null, 2)}</div>;
});
```

or, with `useRef`:

```js
const SomeRefUsingComponent = () => {
  const [position, setPosition] = React.useState(null);
  const ref = React.useRef(null);
  // We pass `ref` to `useScrollPosition`.
  // Note that a `ref` will _not_ be returned.
  useScrollPosition(setPosition, ref);
  return <div ref={ref}>{JSON.stringify(position, null, 2)}</div>;
};
```
